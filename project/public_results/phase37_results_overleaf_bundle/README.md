# Results Overleaf Bundle

This folder is a direct upload bundle for Overleaf.

## Files

- `main.tex`
  Standalone LaTeX document that compiles the Results section by itself.
- `sections/results_section.tex`
  Section-only Results text for insertion into a larger manuscript.
- `tables/`
  Inline tables already referenced from the Results section.
- `figures/`
  Local copies of the Results figures used by the LaTeX file.

## Included content

- Compact journal-style Results section
- Inline figures and captions
- Cohort mean endpoint table
- Full 10-case rank-change table
- Falsification summary table

## Suggested use

1. Upload the entire folder to Overleaf.
2. Set `main.tex` as the main file for a standalone compile.
3. Or copy `sections/results_section.tex`, `tables/`, and `figures/` into an existing manuscript project.
